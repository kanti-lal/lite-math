# Contributing to lite-math

Thank you for considering contributing to lite-math! We appreciate your effort to improve this project.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/yourusername/lite-math.git`
3. Install dependencies: `npm install`
4. Create a feature branch: `git checkout -b feature/your-feature-name`

## Development

- `npm run build` - Compile TypeScript to JavaScript
- `npm run watch` - Watch mode for development
- `npm test` - Run tests
- `npm run test:coverage` - Run tests with coverage report
- `npm run lint` - Run ESLint
- `npm run format` - Format code with Prettier

## Making Changes

1. Write your code following the existing style
2. Add tests for new functionality
3. Ensure all tests pass: `npm test`
4. Run linter: `npm run lint`
5. Format code: `npm run format`

## Submitting Changes

1. Push to your fork
2. Create a Pull Request with a clear title and description
3. Reference any related issues
4. Ensure CI passes

## Code Style

- Use TypeScript for all code
- Follow the existing code style
- Add JSDoc comments for all functions
- Keep functions small and focused
- Write tests for all new features

## Reporting Issues

- Check existing issues first
- Provide clear description of the problem
- Include code examples when possible
- Specify your Node.js and npm versions

Thank you for contributing!
